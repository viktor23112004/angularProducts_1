Для того щоб відкрити проект:
1) потрібно переглянути відео за 31.05.2024р JS Angular;
2) встановити пакет JSON (сайт https://github.com/typicode/json-server);
3) встановлення в терміналі: npm install json-server -g
4) відкрити в терміналі (зайти в папку з базою JSON) cd data
5) ввести: npx json-server db.json
6) в новому терміналі завантажити сервер ng serve --open
